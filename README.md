# lizhi

## 技术栈

`React`, `Vite`, `Chakra-UI`, `react-hook-form`

## 线上演示

[https://lizhi-demo.vercel.app/](https://lizhi-demo.vercel.app/)

## 安装依赖

```sh
npm install
```

## 本地运行

```sh
npm run dev
```
